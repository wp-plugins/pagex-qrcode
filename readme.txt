== Description ==

Awesome QRCode Generator Plugin for WordPress

Generates a QR Code for the current URL. 

Or supply optional attributes for custom text, URL's. 

Make qrcodes for anything. Affiliate links, secret messages, codes, etc

Message attribute can contain any text, custom url

Leave it blank and the qrcode content will contain the current URL of the page or post.


== How to use it ==

* Type the following shortcode into any WordPress Post or Page: 

`[pagex-qrcode]`
  
* If no attributes are supplied a default scan reveals the current URL of the page it was included on.

* To customize the qrcode use optional attributes

`[pagex-qrcode message="" size="160" alt="scan me" align="right" class=""]`

* Optional shortcode attributes:  

- message="Enter custom text or URL" 
- alt="alt text" 
- size="160"
- align="left or right"
- class="cssclass"

== Installation ==

1. Unzip the file pagex-qrcode.zip 
2. Upload the entire `pagex-qrcode` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Then type [pagex-qrcode] into any WordPress page or Post
5. Done
